[start_general_information]
Добро пожаловать в программу TPDatabase!

Общая справка:

 1) Вы можете выбрать нужную вам команду при помощи цифры, соответствующей команде
    и начать Enter.
 2) Вводить нужно только то, что просит программа. Если вы ввели что-то не так,
    то программа сообщит об этом.
 3) Старайтесь перед тем, как производить операции с файлами, убедится, что они открыты.
    Если нужные файлы окажутся закрытыми, то программа сообщит об этом и предложит открыть их.
 4) При редактировании записи документа, новое значение, зависящее от справочника,
    должно существовать в справочнике, иначе изменение отменяется.
 5) При попытке выйти из программы или же при закрытии файла, когда остались несохранённые
    изменения, программа сообщит об этом и предложит сохранить их.
 6) Старайтесь регулярно сохранять свои изменения, чтобы не потерять прогресс.
 7) Удачи!

Введите пункт, по которому вы хотите получить справку при помощи цифр и нажмите Enter:

1. Файл
2. Редактировать документ
3. Вывести на экран документ
4. Записать документ в файл txt
5. Справочник
6. О программе
7. Выйти из справки
[end_general_information]

[start_file]
Файл

В пункте меню файл вы можете выбрать следующие команды:
1. Открыть 
   Открывает файл "tele.db".
   В случае если файл уже был открыт или же его нет, то программа сообщит об этом.
 
2. Сохранить
   Сохраняет изменения всех открытых файлов.
   
3. Закрыть
   Закрывает файл "tele.db". Если файл не открыт, то программа сообщит об этом.

4. Назад
   Возвращается к главному меню.
[end_file]

[start_edit_doc]
Редактировать документ

В этом пункте меню вы можете выбрать следующие команды:
1. Добавить запись
   Добавление записи в документ.
    1) Введите id записи, не совпадающее с существующими.
    2) Введите номер телефона, не совпадающее с существующими.
    3) Вы можете ввести как само ФИО, так и его id, результат будет аналогичен.
       Не забудьте, что новая запись должна существовать в справочнике.
    4) Вы можете ввести как сам тариф, так и его id, результат будет аналогичен.
    5) Введите адрес абонента, не совпадающее с существующими.
    6) Если добавлен справочник "benefit.db", то вы можете ввести как саму льготу,
       так и её id, результат будет аналогичен.
    7) Затем будет выведена добавленная запись.

2. Удалить запись
   Удаление существующей записи из документа.
    1) Для наглядности будет выведен весь документ с записями справочников.
       Введите id той записи, которую вы хотите удалить.
    2) Затем будет выведенная выбранная вами запись, вам нужно нажать "1", если вы
       хотите её удалить и "0", если вы хотите вернуться назад.
    3) Если вы выбрали "1", то запись будет удалена, а если "0" - то вы вернётесь назад.
  
3. Редактировать существующую запись
   Редактирование существующей записи документа.
   Если вы не хотите изменять какую-либо запись, просто нажмите Enter.
   Для наглядности, вам для каждого типа записи будет приведено её текущее значение.
    1) Введите id изменяемой записи, которая должна существовать.
    2) Введите новое id записи, не совпадающее с существующими.
    3) Введите новый номер телефона, не совпадающее с существующими.
    4) Вы можете ввести как само новое ФИО, так и его id, результат будет аналогичен.
       Не забудьте, что новая запись должна существовать в справочнике.
    5) Вы можете ввести как сам новый тариф, так и его id, результат будет аналогичен.
    6) Введите новый адрес абонента, не совпадающее с существующими.
    7) Если добавлен справочник "benefit.db", то вы можете ввести как саму новую льготу,
       так и её id, результат будет аналогичен.
    8) Затем будет выведена откорректированная запись.

4. Назад
   Вовращение к главному меню
[end_edit_doc]

[start_record_doc]
Записать документ в файл txt

Запись всего документа с записями справочников в файл txt.
 1) Введите название файла, в который вы бы хотели записать документ.
 2) Если файл с введённым названием существует, то программа сообщит об этом
    и предложит перезаписать файл или же вернуться назад. Если пользователь решит
    перезаписать файл, то всё его содержимое будет удалено, а на его место будет
    записано содержимое документа с записями справочников. Если пользователь выбрал
    "Назад", то вы вернётесь назад.
 3) Если файл с введённым названием не был обнаружен, то в случае успеха программа
    попытается создать это файл в текущей директории и запишет в него содержимое
    документа с записями справочников и случае успеха сообщит об этом. В случае
    неудачи, программа сообщит об ошибке.
[end_record_doc]

[start_directory]
Справочник

Большая часть справки прошлых аналогичных пунктов справедлива
и для пункта меню "Справочник"
Перед выполнением пунктов "1-5", вы должны ввести тот справочник, с которым вы хотите
работать и должен быть добавлен в программу.
Если вы внесёте какие-либо изменения в справочник, которые затронут связь с документом,
то программа сообщит об этом и предложит восстановить её или же отменить изменение.

В этом пункте меню вы можете выбрать следующие пункты:
1. Открыть справочник
   Открытие справочника, введённого пользователем. Если справочник не известен
   программе, то она об этом сообщает.

2. Редактировать запись в справочнике
   Редактирование существующей записи справочника.
   Если вы не хотите изменять какую-либо запись, просто нажмите Enter.
   Для наглядности, вам для каждого типа записи будет приведено её текущее значение.
    1) Введите id изменяемой записи, которая должна существовать.
    2) Введите новое id записи, не совпадающее с существующими.
    3) Введите новую запись, не совпадающее с существующими.
    4) Затем будет выведена откорректированная запись.

3. Добавить запись в справочник
   Добавление записи в справочник.
    1) Введите id записи, не совпадающее с существующими.
    2) Введите запись, не совпадающую с существуюыщими.
    3) Затем будет выведена добавленная запись.
   После добавление записи будет произведён поиск в документе на соответствие
   добавленному id, id, имеющимися в документе и если такие найдутся, то ссылки будут 
   автоматически им присвоены.
     
4. Удалить запись
   Удаление существующей записи из справочника
    1) Для наглядности будет выведен весь документ с записями справочников.
       Введите id той записи, которую вы хотите удалить.
    2) Если удаляемая вами запись используется в документе, то программа предупредит об
       этом.
    2) Нажмите "1", если вы хотите её удалить и "0", если вы хотите вернуться назад.
    3) Если вы выбрали "1", то запись будет удалена и в документе на месте удалённой
       строки появится запись "(нет данных)", а если "0" - то вы вернётесь назад.

5. Закрыть справочник
   Закрытие добавленного в программу справочника.
    
6. Добавить ещё один справочник
   Добавление справочника "benefit.db".
    1) Введите имя добавляемого справочника "benefit.db".
    2) Если вы правильно ввели название, то программа сообщит, что добавление
       справочника прошло успешно. В этом случае добавленный справочник "benefit.db"
       можно наравне с остальными редактировать, выводить на экран и прочее.
    3) Если вы неверно ввели название добавляемого справочника, то программа сообщит
       об этом и вы вернётесь в меню "Справочник"

7. Вывести на экран
   Этот пункт меню выводит на экран содержимое всех добавленных в систему справочников.
      
8. Назад
   Возвращается к главному меню.
[end_directory]
